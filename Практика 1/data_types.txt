Неизменяемые типы данных — это такие типы,
объекты которых не могут быть изменены после их создания.
Если вы попытаетесь изменить объект неизменяемого типа, то
Python создаст новую ячейку с изменёнными данными, а старая ячейка останется неизменной.

int (целые числа): При изменении создаётся новая ячейка.
float (числа с плавающей точкой): При изменении создаётся новая ячейка.
str (строки): При изменении создаётся новая ячейка.
